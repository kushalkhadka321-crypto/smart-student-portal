Smart Student Portal

-Description:
A full-stack web application to manage student records using Spring Boot and MySQL.

-Features:
Add, update, delete student data (CRUD operations),
MVC architecture,
Backend integrated with MySQL database

-Technologies Used:
Java,
Spring Boot,
MySQL,
HTML, CSS

-How to Run:
*Prerequisites:
Java (JDK 17 or above),
Maven,
MySQL


-Steps
1. Clone the repository
2. Open the project in Visual Studio Code
3. Configure MySQL database in application.properties
4. Run the command in terminal: mvn spring-boot:run
5. Open browser and go to: http://localhost:8080
